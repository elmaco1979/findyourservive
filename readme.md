## Download Instruction

1. Clone the project.

```
git clone https://github.com/infinitypaul/findyourservive.git projectname
```

2. Install dependencies via composer.

```
composer install 
```

3. Migrate and seed the Database.

```
php artisan migrate --seed
```

4. Run php server.

```
php artisan serve
```

Enjoy!
